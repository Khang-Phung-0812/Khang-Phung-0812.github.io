## Run Locally

You can use VS Code Live Server or any static file server.

Example with VS Code:
1. Open the repo folder.
2. Open `index.html`.
3. Start Live Server.

## Contact Form

The contact form posts to Formspree:

`https://formspree.io/f/xjgekzdz`

Make sure this endpoint remains active in your Formspree dashboard.
